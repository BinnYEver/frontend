What is Language Model Net?

Language Model Net (LMNet) is a knowledge graph that can be flexibly extended to any relations. In LMNet, you can define a relation with a template (e.g. "A is the place for B"), and a few example entity pairs (e.g. library, study). Our framework can automatically extract knowledge tuplese from a language model (e.g. DistilBERT). Our framework first gets a set of weighted templates for this relations, and then searchs for the entity pairs that can be naturally fitted in the templates. The results are a list of entity pairs with confidence.
